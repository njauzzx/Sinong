## 数据收集与模型训练
依托南京农业大学学科特色，面向**动物科学**、**动物医学**、**农业经济管理**、**农业资源与环境**、**园艺学**、**智慧农业**、**植物保护**、**作物育种**等细分学科收集了大规模领域数据，总量超40亿token。
|数据类型|数据量|
|:---:|:---:|
|书籍|8,863册|
|论文|243,897篇|
|政策、标准、专利等|196,748份|
|||

结合视觉模型与大语言模型，这些多源异构的数据被整理、筛选、整合，形成了**大规模**、**高质量**的**农业基础数据集**。

为提升模型的农业语言能力与领域知识，我们在基础数据集上开展了合成数据构建、指令微调与强化学习，除传统指令数据外，思维链、上下文参考形式的指令数据也被应用于模型训练中，旨在提升模型对农业领域知识的理解、学习和使用能力。

### 多智能体检索增强
当前，检索增强（RAG）已经成为大语言模型的领域应用不可或缺的一环，然而，当前针对领域专业文献（数据论文）的检索增强仍存在局限性。我们推出了一套多智能体检索增强框架，针对知识库构建、数据来源、检索效率等方面进行了优化。
